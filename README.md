# iccs205-gym-project
Main Functions (Screens)

Each one will have PL/pgSQL functions:

1. Membership Management

- Add a member

- Assign membership type

- Update membership

- Check membership validity

- Get list of expiring memberships

2. Payment Processing

- Insert new payment

- Calculate outstanding balance

- Get payment history

- Auto-renew membership (PL/pgSQL function)

3. Equipment Maintenance

- Add equipment

- Log maintenance

- List equipment needing maintenance

- Track usage count per machine

4. Progress Tracking

- Insert weight/body fat log

- Fetch progress over time

- Calculate attendance frequency
